При запуске батника "создать слайдшоу" запускается программа. Необходимо ввести ссылку на папку, содержащую или картинки, или папки с картинками. 
Потом указывается количество кадров для слайдшоу.
После этого программа разбивает все картинки в папке по подпапкам, и создает из них слайдшоу используя ffmpeg.
